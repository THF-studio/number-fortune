# 号运 Roadmap

## v1.0 — Current (Core Analysis)

Single phone number analysis with full I Ching pipeline.

- Phone number input with digit parsing and zero-replacement
- Three-layer hexagram display: 本卦 / 互卦 / 之卦
- Five-element layer table with 体/用 relationship
- Golden rule verdict card (overall pass/fail)
- Digit-by-digit element chips
- Per-focus-area score cards (family, health, study, career, wealth)
- 喜忌 (favored/avoided element) toggles
- React Router: `/` (analysis) + `/bulk` (placeholder)

---

## v1.1 — Next

- **Compare two numbers**: side-by-side analysis of two phone numbers
- **Export PDF**: download a formatted analysis report
- **Share result as image**: generate a shareable card (canvas or html-to-image)

## v2.0 — Future
